==== leetcode.15.三数之和 ====
* 题目描述：
给你一个整数数组 nums ，判断是否存在三元组 [nums[i], nums[j], nums[k]] 
满足 i != j、i != k 且 j != k ，同时还满足 nums[i] + nums[j] + nums[k] == 0 。
请你返回所有和为 0 且不重复的三元组。

注意：答案中不可以包含重复的三元组。

* 输入描述：
第一行输入一个整数N，数组长度；
第二行N个整数，数组元素。


* 输出描述：
符合条件的三元组，不含重复项。


==== 示例 ====
示例1：
输入：
6
-1 0 1 2 -1 -4

输出：
-1 0 1
-1 -1 2

------------------------
示例2：
输入：
3
0 1 1

输出：
NULL

------------------------
示例3：
输入：
3
0 0 0

输出：
0 0 0

============
解题：先排序，固定一个指针 从该指针右侧左右移动另外两个；
移动过程中跳过重复变量
通过率：100%
